### 7.1 Built an agentic "team brain" ("Brian")
Designed a **per-user, privacy-first, daily-compounding agentic knowledge base** for solution-engineering teams, combining two cutting-edge 2026 patterns — Andrej Karpathy's *LLM Wiki* and Tobi Lütke's *QMD* local retrieval engine. The system:
- runs **fully local** (on-device inference, no cloud dependency) — privacy by architecture, designed to meet enterprise data-governance expectations without IT policy enforcement;
- uses a **~280-line schema** (routing rules, page templates, ingest/query/lint operations, bilingual frontmatter) governing how raw sources (calls, transcripts, chat, calendar, CRM, email) become a linked wiki of *sources → entities → concepts → synthesis*;
- runs **scheduled, zero-touch daily ingest** so the knowledge base maintains itself and compounds;
- exposes the corpus to agents via an **MCP server** for natural-language retrieval.
He named it **"Brian"** (a person's name + an echo of *brain*) and **presented it at an internal PayPal AI innovation contest** — moving it from personal experiment to organizational visibility.

### 7.2 Production agentic workflow automation (DOE framework)
Implemented the **Directive–Orchestration–Execution** pattern (Markdown SOPs → an LLM orchestrator → deterministic Python execution scripts) to solve the "compounding-failure" reliability problem in multi-step AI workflows. He ran a **production multimodal pipeline** (local Gemma model analysing video → structured Markdown knowledge base) at meaningful scale, then **proposed and demoed** applying the same architecture to solution-engineering operations (automated activity reports and partner-status synthesis from chat, email, and call transcripts).

- **MCP as connective tissue** — daily, production use of MCP connectors across knowledge, collaboration, CRM, and BI tools — including **diagnosing and fixing** auth/transport and corporate TLS-inspection failures rather than abandoning the tool.
- **Self-serve analytics** — pulls business metrics (e.g. trailing-12-month Net TPV, YoY) directly from an enterprise BI semantic layer via MCP, including entity disambiguation and query construction.
- **AI-consumable documentation** — deliberately structures SDDs so partner coding agents can ingest and act on them.
- **Multi-format, brand-compliant generation** — programmatic `.docx` / `.pptx` / HTML / Markdown / CSV / Mermaid output with citation discipline.
